# Room1 (Networking)

## Overview
This module introduces the basic concepts of computer networking, including how devices communicate within and across networks. It covers addressing systems, protocols, and identity in networking.

## Key Concepts Learned

### What is Networking?
- Networking is the process of connecting multiple devices to share data.
- The Internet is a global network that connects multiple smaller networks together.

### IP Address (Internet Protocol)
- Communication over networks follows strict rules called protocols
- IP stands for Internet Protocol
- Used to identify a device on a network

### IPv4
- Consists of 4 sections (octets)
- Each octet ranges from 0–255
- Example: 192.168.1.1

### IPv6
- Newer version of IP addressing
- Created due to the shortage of IPv4 addresses
- Supports a much larger number of devices

### Public vs Private IP
### Private IP
- Used inside a local network (home, school, etc.)
- Not directly accessible from the internet
### Public IP
- Used to communicate over the internet
- Assigned by ISP

### MAC Address (Media Access Control)
- A unique identifier assigned to a network interface
- Consists of 12 hexadecimal characters (6 sections)
- Example: 00:1A:2B:3C:4D:5E
- First 3 sections (OUI) → Identify the manufacturer
- Last 3 sections → Unique to the device

### MAC Spoofing
- MAC addresses can be changed (spoofed)
- Used to bypass certain security controls like filters or firewalls

### Device Identity Concept
Devices have two types of identity:
- IP Address → Changeable (like a name)
- MAC Address → Hardware-based (like a fingerprint)
- (Note: MAC can technically be spoofed, so it's not 100% permanent — but it's still more hardware-tied than IP)

## Skills Gained
- Basic understanding of networking concepts
- Identifying IP and MAC address structures
- Understanding network identity and communication
- Awareness of basic security concepts like spoofing
